# 🧵 Bhagya Tailor Billing App

A simple, modern, and responsive billing web app built using **HTML, CSS, and JavaScript**.

Designed for small businesses like tailoring shops to quickly calculate prices, discounts, and totals.

---

## ✨ Features

* 📱 Mobile-friendly and clean UI
* 🧾 Add multiple product entries dynamically
* 💰 Automatic price and discount calculation
* 🔢 Quantity-based total calculation
* 📊 Total summary with savings highlight
* 📷 Image upload with preview
* 📄 Download bill as PDF
* 📸 Integrated Instagram contact link
* 💜 Beautiful theme using **#5e17eb**

## 📦 How to Use

1. Enter product details
2. Add price, discount, and quantity
3. Add multiple items if needed
4. View total amount and savings
5. Click **Download as PDF** to generate bill

---

## 💡 Use Case

This app is useful for:

* Tailors
* Small shop owners
* Local businesses needing quick billing

---

## 📁 Deployment (GitHub Pages)

1. Upload the HTML file to your repository
2. Go to **Settings → Pages**
3. Select branch → `main`
4. Your app will be live
